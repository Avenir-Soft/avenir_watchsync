# Avenir WatchSync

> Смотришь фильм в браузере на компе — продолжаешь с той же секунды на телефоне. И наоборот. Локально, дома, без облака, ноль вложений.

Синхронизация позиции видео между устройствами в домашней WiFi. Работает на **любом сайте** (онлайн-кинотеатры, пиратки) — ловит `<video>` даже внутри вложенных iframe и «закрытого» Shadow DOM плееров.

## Как устроено

```
┌────────────┐   позиция    ┌──────────────────────┐   позиция   ┌────────────┐
│  ПК Chrome │ ───────────► │  Сервер (Docker)     │ ◄────────── │ Телефон    │
│ расширение │              │  FastAPI + SQLite    │             │ Mises +    │
│            │ ◄─── перемотка│  + дашборд на :8765  │ перемотка ─►│ расширение │
└────────────┘              └──────────────────────┘             └────────────┘
                    всё в одной домашней WiFi
```

- **Сервер** — FastAPI + SQLite в Docker. Хранит позиции и постеры, отдаёт дашборд.
- **Расширения** (MV3) — content-script находит `<video>` на любой странице, шлёт позицию в фон → на сервер, и при открытии перематывает.
- **Дашборд** `http://<ip>:8765` — «продолжить просмотр»: постеры, прогресс, избранное, поиск/фильтр, отправка в Telegram.

## Запуск сервера

```bash
cp .env.example .env          # опционально: впиши TG_TOKEN для Telegram-кнопки
docker compose up -d
```

- ПК: `http://localhost:8765`
- Телефон (та же WiFi): `http://<LAN-IP-компа>:8765`
- Автозапуск: `restart: unless-stopped` (поднимается со стартом Docker). На Windows — ярлык `start-server.bat`.

## Расширение — ПК (Chrome)

`chrome://extensions` → включить **Режим разработчика** → **Загрузить распакованное** → папка `extension/`.

## Расширение — телефон (Mises / Kiwi / Lemur)

Обычный Chrome на Android расширения **не поддерживает** — нужен Chromium с загрузкой расширений.

1. Play Market → **Mises Browser**.
2. Открыть `http://<LAN-IP>:8765/kiwi.zip` (скачается сборка).
3. Меню → **Extensions** → **Developer mode** → **＋** → выбрать zip.

> Адрес сервера для телефона задан в `extension-phone/background.js` (`SERVER = http://<LAN-IP>:8765`). При смене IP — поправить и пересобрать zip.

## Telegram (опционально)

Кнопка ✈ на карточке шлёт ссылку «продолжить» тебе в личку.

1. @BotFather → `/newbot` → скопировать токен в `.env` (`TG_TOKEN=...`).
2. Написать своему боту любое сообщение (chat_id определится автоматически).

## Возможности дашборда

Постеры (og:image) · прогресс-бар · ⭐ избранное · 🔎 поиск · фильтр (все / в процессе / избранное) + сортировка · 🔴 «смотрят сейчас» · секция «Просмотрено» · отправка в Telegram · ярлык на телефон (PWA) · монохромный UI, светлая/тёмная адаптация.

## Структура

```
server.py             FastAPI: REST API + дашборд (одним файлом)
Dockerfile            образ python:3.12-slim
docker-compose.yml    порт 8765, том ./data, TG_TOKEN из .env
extension/            расширение для десктопного Chrome
extension-phone/      расширение для телефона (раздаётся как /kiwi.zip)
start-server.bat      быстрый запуск на Windows (WSL + Docker)
data/                 том с БД watch.db (в .gitignore)
```

Ключевые приёмы: `content_scripts` c `all_frames:true` + рекурсивный обход Shadow DOM; патч `attachShadow` (`world:"MAIN"` + инжект-фолбэк) для «закрытых» плееров; длительность через `seekable.end()` когда плеер не отдаёт `duration`; нормализация URL (срез utm-меток) чтобы одна ссылка = одна карточка.

## Ограничения

- **Телефон** — только браузер с расширениями (Mises/Kiwi/Lemur), не обычный Chrome.
- **Сериалы** на сайтах с одним URL на все серии (серия переключается внутри плеера) — восстановление конкретной серии зависит от плеера (для kodik реализуемо).
- Отдельные «изолированные» плееры прячут `<video>` в недостижимый sandbox-iframe — обход: выбрать другой плеер на странице.
